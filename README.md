# music_mixer_assigement
This is a Repository for the Music Mixer assignment for Multi Media Authoring 2.

## Installation 

No installation required. Download the zip file and extract.

## Usage 

Open index.html in the browser of your choice, click and drag instruments and click the play button to play music mixed from the instruments.
